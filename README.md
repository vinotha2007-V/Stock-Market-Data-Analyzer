# 📈 Stock Market Data Analyzer

## 📌 Overview
This project analyzes stock market data using Python. It fetches live stock prices, calculates technical indicators, visualizes trends, and predicts future prices using Machine Learning.

---

## 🚀 Features
- Real-time stock data
- Technical indicators
- Data visualization
- Price prediction
- Interactive dashboard
- Financial analysis

---

## 🛠️ Technologies
- Python
- Pandas
- NumPy
- Streamlit
- Plotly
- Scikit-learn
- yfinance

---
